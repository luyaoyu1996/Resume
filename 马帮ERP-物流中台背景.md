# 马帮镖局物流中台 - 背景信息

## 产品定位

马帮镖局是马帮 ERP 的物流中台，连接 ERP 系统与物流商/海外仓，承担跨境电商订单的物流履约全链路。

## 核心业务模块

### 1. 直发物流
- 承接 ERP 订单，下发给物流商
- 回写运单号和面单到 ERP
- 接入直发物流商 **1800余家**

### 2. 海外仓履约
- 海外仓商品、库存同步
- 商品创建
- 仓配订单履约
- 仓配分离订单履约
- 接入海外仓 **1000余家**

### 3. Open API
- 主动对接：镖局主动对接物流商/海外仓接口
- 被动对接：物流商/海外仓对接镖局接口（双向）

## 规模数据
| 维度 | 数量 |
|------|------|

| 接入海外仓 | 1000+ |
