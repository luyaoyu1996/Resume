# BOSS 直聘打招呼策略库

> 跨境电商 SaaS + AI 产品方向的打招呼话术体系
> 核心原则：**4 锚精准打击 + JD 关键词反射 + 数字脱敏**
> 持续迭代：每次实战后回写案例和教训

---

## 一、核心方法论：4 锚结构

HR 看打招呼的**前 5 秒**就决定是否回复。4 锚对应 HR 5 秒内必问的 4 个问题：

| # | 锚 | 对应 HR 问题 | 你要给的答案 |
|---|---|-------------|-------------|
| 1 | **身份锚** | "这人是谁？" | 5 年 SaaS B 端 + 当前方向 |
| 2 | **硬数据锚** | "做过什么规模？" | 月订单亿级 / 1800+ 物流商 / 年营收数百万 |
| 3 | **差异化锚** | "凭什么是你？" | 产品+技术双栈 / AI 0→1 全栈 / BEC-H 英语 |
| 4 | **针对性锚** | "为什么投我们？" | **反射 JD 原文关键词** + 动机 |

**缺 1 锚 HR 会犹豫，缺 2 锚直接跳过。**

## 二、按 JD 类型选打法

### 🟢 类型 A：JD 明确要求 AI Agent / LLM / 工作流

**打法**：**AI 技术密度拉满**
- OpenClaw 五层架构 + ReAct 模式 + "AI 判断 / CLI 执行"
- Echo RAG + Dify 工作流编排
- **JD 里的术语（ReAct / 任务分解 / 工具调用 / 记忆）主动复述 2-3 个**

**案例**：叮咚买菜 AI 产品经理（罗宁）

---

### 🔵 类型 B：JD 聚焦跨境电商履约 / 物流 / 海外仓

**打法**：**跨境履约深度打满 + 英语加分**
- 月订单亿级 + 1800+ 物流商 + 1000+ 海外仓
- 保税仓 / 批次追踪 / 三层闭环 / 仓间中转调拨
- **BEC-H 188 英语必须打出来**（干掉 80% 无英语竞品）
- AI 放最后当锦上添花

**案例**：得物国际供应链产品经理（袁女士）

---

### 🟡 类型 C：JD 偏通用 B 端 / 商家产品 / 方向不定

**打法**：**双锚切入**（主打能力 + AI 钩子）
- 主锚选你最硬的战场（履约 / 商家产品 / 开放平台）
- AI 作为"钩子"主动接 JD 里"AI 场景应用"这类措辞

**案例**：小红书 B 端产品经理通用招募（潘雨洁）

---

### 🔴 类型 D：JD 品牌方 / 卖家方 / 运营提效

**打法**：**业务理解展示 + AI 落地方法论**
- 跨境业务 7 环节全景（选品 / Listing / 美工 / 采购 / 运营 / 履约 / 售后）
- OpenClaw 现场跟单访谈方法论
- 突出"降低非技术人员使用门槛"

**案例**：衡哲信息跨境电商 AI 产品经理

## 三、JD 关键词反射技巧

**HR 的潜意识**：看到自己 JD 原文出现在打招呼里 → "这人真的看过 JD"。

### 反射词典（按 JD 类型）

| JD 关键词 | 你的反射话术 |
|----------|-------------|
| "AI 场景和应用" | "看到 JD 里'结合 AI 场景',正是我过去一年独立 owner 的方向" |
| "ReAct / 任务分解 / 工具调用" | "JD 里提到的 ReAct、任务分解、工具调用,正是 OpenClaw 过去一年每天在打磨的能力" |
| "降低非技术人员使用门槛" | "OpenClaw 给采购员、Echo 给 PM/客服,都是非技术人员日常用" |
| "跨境进出口 / 保税 / 海外仓" | 把马帮对应业务精准对上 |
| "商家产品 / 降本增效" | 头部客户定制 + 降本增效产品群 |
| "0→1 / 从 0 到 1" | "2 款企业级 AI 产品 0→1 独立全栈" |
| "英语工作语言" | "BEC-H 188 分,具备全英文项目交付能力" |
| **"数据驱动 / 降本增效 / 数据分析"** | **"智能优选渠道从'人工弹窗逐单选'升级为系统自动匹配,月服务百万+订单,每单省 $5-$40,整体承运成本下降 12-18%"** |
| **"AI 辅助 / 技术栈 / 产品技术双栈"** | **"日常 Claude Code/Cursor 做 AI 辅助开发,可独立完成功能 MVP 验证上线"** |
| **"补货 / 库存优化 / 智能计算"** | **"规划智能计算补货板块,积压/断链比率降低 30~50% 不等"**（待补详情） |

### 数据驱动标准句式（★ 固化为标配）

**4 段式结构**：场景对比 + 规模数字 + 单位节省 + 整体 ROI

> 智能优选渠道从**"人工弹窗逐单选"升级为系统自动匹配**，**月服务百万+订单**，**每单省 $5-$40**，**整体承运成本下降 12-18%**。

**为什么这一句最好用**：
- **场景对比**"人工弹窗逐单选 → 系统自动匹配" = HR 一秒共鸣痛点
- **规模数字**"月服务百万+订单" = 证明不是小样本
- **单位节省**"每单省 $5-$40" = 可验证的硬数字（不是百分比）
- **整体 ROI**"整体承运成本下降 12-18%" = 从客户账单同比反推,经得起追问

**使用规则**：
- 凡 JD 出现"数据驱动/降本增效/数据分析/业务结果/A/B 测试/漏斗优化"→ **必须用这一句**
- 不要拆开用,4 段一起报才有密度
- 后面可补"好产品要跟上业务节奏"作为方法论收尾

## 四、通用模板骨架

```
【称呼】老师您好!卢耀煜,5 年 SaaS B 端产品经验,【针对 JD 的一句精准定位】:

1、【主锚 1:最硬的 1:1 match,用数据砸】

3、【差异化锚:AI 双栈 / 英语 / 技术双栈 选 1-2】

4、(可选)【补充锚:JD 有但你不是强项的,轻量回应】

【收尾钩:反射 JD 原文关键词 + "期待聊聊~"】
```

**3 锚最精简,5 锚最长限**——再多 HR 会累,再少打不出密度。

## 六、避雷清单

### 通用避雷
- ❌ "听说贵司发展很好" — 空话
- ❌ "期望贵司给我成长机会" — 显弱势
- ❌ 主动问薪资/地点/是否出差 — HR 第一轮太早
- ❌ 加 emoji 表情 — B 端岗位不合适
- ❌ 话术超过 6 锚 / 字数超 300 — HR 会跳过

### 按岗位类型避雷
| 岗位 | 不要提 |
|------|-------|
| AI 产品岗 | 太多非 AI 的战场数据 |
| 物流履约岗 | 年营收 700 万(HR 不 care) |
| 国内业务岗(叮咚/小红书国内) | BEC-H 英语 / 跨境海外仓 |
| 跨境业务岗 | AI 双栈放第 1(应放第 5) |
| 品牌方 AI 岗 | 过度强调 SaaS 经验 |

### 案例 10:呜呜很忙集团 供应链资深 PM(OMS)(罗哲,高招专家,26 分钟前回复,今日 3 次)

**JD 特点**:**OMS 订单中心资深产品岗**,JD 完全不提 AI,核心要求 OMS+WMS/TMS 集成+智能分仓+跨仓调拨+缺货处理+中台重构,5-10 年经验+3 年 OMS 负责人。**这是所有岗里 JD 和你素材对应最精确的一家**——每条要求都有直接素材,无需桥接。

### 案例 11:货拉拉 跨境物流解决方案岗(林先生,资深 HRBP,9 分钟前回复,今日 10+ 次)

**JD 特点**:**不是产品岗,是解决方案/KA 大客户岗**——售前协同销售+海外客户需求调研+招投标+方案标书+成本把控+项目实施。优先 3PL 履历(嘉里/顺丰/利丰/京东物流)。**薪资 20-40K 无 15/16 薪,年包上限 48W**。

**打法**:**解决方案架构师视角 + 头部客户对接闭环经验 + 英语 BEC-H 188 + 淡化产品 PM 标签**。

**匹配度**:70%(销售/招投标/3PL 履历 3 个硬短板,但物流深度+头部客户对接可部分弥补)

**反射词**:"需求调研、方案共创、项目实施、运营落地"(解决方案岗动词),"仓配一体+国际物流+供应链解决方案"(JD 原文)

**关键话术**:
- **"这套 KA 大客户对接+方案设计+成本核算+交付闭环的能力可以迁移到贵司售前场景"** — 主动说"可以迁移",不装做过
- **"我站在另一侧——从 SaaS 和货主视角看 3PL"** — 化解 3PL 履历缺失

**风险点**:
- **销售/招投标经验零** → 话术里只讲头部客户对接能力,不吹售前售后;坦诚"标书制作是需要补强部分"
- **3PL 履历缺失** → "从 SaaS 侧看 3PL 是不同维度"化解
- 出差陪客户节奏可能不适合产品岗习惯
- 薪资上限低于字节/得物等顶级岗

**这家核心打法特点**(非产品岗模板):
- 把产品经理身份重新定位成"解决方案对接者",换一套动词体系
- 头部美妆百万级项目作为"类售前→方案→交付闭环"的类比证据
- 英语 BEC-H 188 必须亮出(海外客户)
- AI/技术栈完全不提(跨岗位类别)

**重要信号**:货拉拉 D 轮+10000 人大厂,但薪资不如字节/得物 → 岗位内部优先级可能不高 → **投了试水,期望值低,不过度准备**

## 十、打招呼 → 初面 衔接技巧

**对方回复"可以聊聊"之后的第一句**:

1. **重要别先问薪资**——显得功利
2. **直接主动抛钩**:"您这边方便先简单聊聊贵司这个岗位现在最紧迫的 1-2 个场景吗?我好有针对性展开"
3. **或者先对方**:"麻烦问下您这边约电话方便的时间段?"

**避免打"组合拳"**:"您好 + 我的情况 + 我想问的 + 我期望的"—— 一次性怼过去显得急切。

**节奏**:先约时间 → 电话/视频里再展开 → 面试/二面再谈细节。
