### 1、接口描述

- API 方法 api.biaoju.order.find

- encodeParams 数据 JSON 结构实例

  A.通过多条件查询订单
```json
    {
        "dateFrom":"订单创建时间, 查询开始时间", 
        "dateTo":"订单创建时间,查询结束时间", 
        "status":"订单状态",
        "page":" 分 页 参 数 ， 第 几 页 ", 
        "rowsPerPage":"分页参数，每页最多显示"
    }
```

  B.获取待处理的订单
```json
    {
        "myNotify":"这里固定传值 1"
    }
```

  C.通过镖局物流单号查询订单
```json
    {
        "code":"内部订单号"
    }
```

   D.通过多个镖局物流单号查询订单
```json
    {
        "codes":"内部订单号,内部订单号,内部订单号"
    }
```

### 2、字段说明

| 字段名称 | 字段类型 | 默认值 | 必填 | 备注 |
| ------ | --------| ----- | ----| -----|
| dateFrom | datetime |  | false |订单创建时间,查询开始时间，默认近10 天<br>数据可选格式：(2015/01/01, 20150101, 2015-01-01) |
| dateTo | datetime |  | false |订单创建时间,查询结束时间，默认当天<br>数据可选格式：(2015/01/01, 20150101, 2015-01-01) |
| status | string |-1 |false|订单状态，可选状态<br>2待入库<br>3已入库<br>4已出库<br>5已完成<br>6已确认（供应商已下载）<br><br>-1 异常订单<br>（订单不符合供应商要求）|
| page| string| 1| false|分页参数，第几页| 
| rowsPerPage| string|100 | false|分页参数，每页最多显示100| 
| myNotify| string| | true|获取通知合作商的订单| 
| code| string| | true|镖局物流单号<br>例：891516800021725| 
| codes| string| | true|多个镖局物流单号,多个单号以英文逗号(,)隔开<br>例:891516800021725,891516600021720|
