## 🎯 3. Y2 下 call 7 档故事（必背 · 骨架版）

**4 步讲完**：

1. **背景**：2024 中美关税战 → 卖家从直发转 Y2 货代（国内仓→机场→海外仓→等 Temu 6 天面单→下发海外仓）
2. **问题**：原版"交运头程立即下 call" 三大缺陷 → **信息不准 / 头程被退 / 48h 冲动退款**
3. **迭代**：深度访谈核心卖家后，改为**7 档时间枚举**（交运立即 / ERP 发货立即 / T+2/T+3/T+4 / 平台可打面单时 / 不自动提交）+ **兜底不晚于平台面单时间**
4. **收尾**：这个项目让我最有感触的是——**产品判断来自现场访谈，不是拍脑袋**

## 🎯 4. 4 个高风险追问（必背 · 反射式）

### 🔴 "你做物流，怎么做选品/Listing/美工？"

> **上游跨境生态天天对接**。OpenClaw 采购员跟单 2 周 → 供应商管理熟；头部客户从美妆到 3C 全品类 → 选品/Listing 节奏熟。**业务逻辑熟，上手不会慢**。

### 🔴 "你 AI 是内部工具，我们要业务 AI"

> **OpenClaw 给采购员用 / Echo 给 PM/客服/开发用，都是非技术人员**。**降低非技术门槛** 是我做这 2 款的核心命题，和 JD 完全一致。

### 🔴 "我们用 n8n/coze/影刀，你用 Dify"

> **Dify ≈ n8n ≈ coze，底层都是节点+变量+LLM+API**。Echo 的 Dify 工作流我是深度用过的，**迁移无本质障碍**。影刀我入职前自学。

### 🔴 "你觉得跨境 AI 最大挑战是什么？"

> **不是模型能力，是场景抽象 + 工作流设计**。**AI 可执行最小单元**（拆细准确率才高）+ **人工 Review 兜底**（一次批量改价错就是几十万损失）。

## 🎯 8. 术语急救包（10 个最高频）

| 词 | 1 秒解释 |
|---|---------|
| **Listing** | 亚马逊商品详情页整体 |
| **五点描述** | Listing 5 条核心卖点 |
| **A+ 页面** | 品牌注册后的富文本图文描述 |
| **MOQ** | 最小起订量 |
| **ACOS** | 广告花费/销售额（越低越好） |
| **FBA / FBM** | 亚马逊自营 / 卖家自发 |
| **Y2** | 关税战下的货代转运模式 |
| **ASIN** | 亚马逊 SKU 唯一 ID |
| **RAG** | 检索增强生成（Echo 用的） |
| **AI Agent** | 会调工具的 AI（OpenClaw 用的） |

**不确定怎么办**：**用通俗讲法代替**（"五点描述" → "5 条核心卖点"），**千万别硬装错**。
